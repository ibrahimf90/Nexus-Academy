# Nexus Academy

Nexus Academy is a premium, beautifully crafted digital bookshelf designed to empower developers with high-quality learning resources. Built from scratch with a focus on modern aesthetics and seamless navigation, the platform provides a structured path for mastering both Front-End and Back-End development.

## Features
- **Path-Based Learning**: Dual curricula designed to take you from UI/UX fundamentals to complex server-side systems.
- **Curated Development Library**: A collection of high-quality development guides including HTML, CSS, JavaScript, React, PHP & MySQL, and Python.
- **Premium User Experience**:
  - **Glassmorphism Design**: High-end visuals using backdrop filters and subtle transparencies.
  - **Dynamic Backgrounds**: Smoothly animated gradient blobs that create a living, breathing interface.
  - **Scroll Reveal Animations**: Content that gracefully enters the viewport as you explore.
- **SEO Optimized & Responsive**: Fast-loading semantic structure that looks stunning on desktops, tablets, and mobile devices.

## Technologies Used
- **HTML5**: Semantic structure for a robust and accessible web foundation.
- **CSS3 (Vanilla)**: Advanced styling featuring CSS Variables, complex gradients, glassmorphism, and custom keyframe animations.
- **JavaScript (Vanilla)**: Powers the interactive elements, including Intersection Observer for scroll animations, navigation state management, and smooth anchor scrolling.
